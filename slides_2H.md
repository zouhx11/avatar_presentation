# How We Train a Talking Photo (Intuitive)

</div>

:: left ::

<div class="ns-c-tight">

## Core Terms
- **Dataset**: many examples to learn from
- **Model**: neural network that maps input to output
- **Loss**: error score (lower is better)
- **Epoch**: one full pass over the dataset
- **Inference**: using trained model on new input

## Mental Model
- Training is like coaching: predict, compare, correct, repeat.

</div>

# Loss Functions: How the Model Knows It Is Improving

</div>

:: left ::

<div class="ns-c-tight">

## Intuitive Loss Terms
- **Lip-sync loss**: do lips match audio?
- **Identity loss**: does the face still look like the same person?
- **Temporal loss**: are motions smooth (no jitter)?
- **Visual quality loss**: are frames sharp and realistic?

## One-Line View
- Total loss is a weighted mix of these scores.

</div>
