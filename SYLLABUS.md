# AI Avatar Workshop Syllabus
## Build Your Own Talking Avatar App! | 2 Hours

**Instructor:** Prof. Guang Cheng, UCLA Trustworthy AI Lab
**Audience:** High School Students | **Prerequisites:** Basic computer skills

---

### Course Goal
Build a working AI-powered talking avatar chat application from scratch.

---

### Schedule Overview

| Class | Topic | Time | What You'll Do |
|:-----:|-------|:----:|----------------|
| **1** | AI Foundations & Python | 25 min | Learn how AI thinks, write your first chatbot |
| **2** | Web Apps & Deep Learning | 25 min | Build web apps with Streamlit, connect to AI APIs |
| **3** | Avatar Technology | 25 min | Explore how talking avatars work, try avatar generators |
| **4** | Integration & Features | 25 min | Connect everything, add memory & error handling |
| **5** | Demo Day | 20 min | Team projects, presentations, awards |

---

### Key Skills You'll Gain

| Technical | Creative |
|-----------|----------|
| Python programming | UI/UX design |
| API integration | Problem-solving |
| Neural network basics | Team collaboration |
| Web development | Technical presentation |

---

### Tools & Resources

| Tool | Purpose |
|------|---------|
| [Google Colab](https://colab.research.google.com) | Python coding environment |
| [Streamlit](https://streamlit.io) | Web app framework |
| [Claude API](https://docs.anthropic.com) | AI chat functionality |
| [Hugging Face](https://huggingface.co/spaces) | Avatar demos |

---

### What You'll Build

```
User Message → AI Brain (Claude) → Text Response → Avatar API → Talking Video
```

**Final Product:** A web app where users chat with an AI that responds through a talking avatar!

---

### Assessment

| Level | Requirements |
|-------|--------------|
| Bronze | Working chat interface |
| Silver | Chat + avatar integration |
| Gold | Full app with custom features |
