# AI 虚拟形象工作坊教学大纲
## 打造你的专属 AI 数字人！| 2 小时

**讲师:** 程光 教授，UCLA 可信人工智能实验室
**学员:** 高中生 | **前置要求:** 基础计算机操作

---

### 课程目标
从零开始构建一个 AI 驱动的会说话的数字人聊天应用。

---

### 课程安排

| 课时 | 主题 | 时长 | 学习内容 |
|:---:|------|:---:|----------|
| **1** | AI 基础与 Python | 25 分钟 | 了解 AI 原理，编写第一个聊天机器人 |
| **2** | Web 应用与深度学习 | 25 分钟 | 用 Streamlit 搭建网页应用，连接 AI 接口 |
| **3** | 数字人技术 | 25 分钟 | 探索数字人原理，体验数字人生成器 |
| **4** | 系统集成与功能完善 | 25 分钟 | 整合所有模块，添加记忆与错误处理 |
| **5** | 项目展示日 | 20 分钟 | 团队项目、成果展示、颁奖 |

---

### 核心技能

| 技术能力 | 综合素养 |
|---------|---------|
| Python 编程 | 界面设计 |
| API 接口调用 | 问题解决 |
| 神经网络基础 | 团队协作 |
| Web 开发 | 技术演讲 |

---

### 工具与资源

| 工具 | 用途 |
|-----|------|
| [Google Colab](https://colab.research.google.com) | Python 编程环境 |
| [Streamlit](https://streamlit.io) | Web 应用框架 |
| [Claude API](https://docs.anthropic.com) | AI 对话功能 |
| [Hugging Face](https://huggingface.co/spaces) | 数字人演示 |

---

### 你将构建的系统

```
用户消息 → AI 大脑 (Claude) → 文本回复 → 数字人 API → 说话视频
```

**最终成果:** 一个网页应用，用户可以与 AI 对话，AI 通过会说话的数字人形象进行回复！
